# Full-Stack MERN final project

## Run server 

Change directory using `cd` to `server`.\
First install `node_modules` using `npm i / npm install`.\
Create .env file using the given example.\
Run command 1npm run server`

## Open up a new terminal and run client side page

Change directory using `cd` to `client`.\
First install `node_modules` using `npm i / npm install`.\
Run command `npm start`
